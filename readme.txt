Esse é um projeto de um banco chamado Bytebank, criado na linguagem Java.

Foram criados duas classes: uma para a criação de um cliente e outra para a criação de uma conta. 

Na classe cliente, é possível inserir o nome do cliente, seu CPF e sua profissão.

Na classe conta, você precisa de uma cliente criado com a classe cliente e lá é possível criar uma agência e um número para a conta. 

Através dos métodos da classe conta é possível sacar, depositar e transferir um valor a partir da conta de um cliente.
